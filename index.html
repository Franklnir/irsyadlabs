Server High-Availability (HA)

Infrastruktur 2 node Ubuntu dengan Keepalived (VRRP) untuk failover otomatis dan HAProxy sebagai reverse proxy/load balancer. Cocok untuk web/API yang butuh uptime tinggi.

Fungsi Utama
  • Failover otomatis: Virtual IP pindah ke node backup saat node utama down.
  • Load balancing request ke backend (round-robin/leastconn sesuai kebutuhan).
  • Health-check backend (tcp/http) supaya request tidak nyasar ke server sakit.
  • TLS termination + hardening header (opsional) untuk production.
  • Minim downtime saat maintenance: tinggal drain traffic, update, balik lagi.
Ubuntu HAProxy Keepalived TLS UFW MySQL
